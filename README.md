# ฅ(ﾐ⚈ᆽ⚈ﾐ)ฅ Blender Hotkeys Helper
Telegram-бот для изучения горячих клавиш Blender с использованием искусственного интеллекта.

## Описание
**Blender Hotkeys Helper** — это интерактивный Telegram-бот, который помогает пользователям запоминать горячие клавиши в Blender. Бот использует нейросеть для генерации уникальных вопросов, адаптированных под конкретные действия и разделы программы.

### Использовано 6 разделов Blender:
- Моделирование
- Текстурирование
- Скульптинг
- UV-развёртка
- Риггинг
- Анимация

### После каждого ответа можно выбрать следующие действие:
- Новый вопрос по текущей теме
- Смена раздела
- Завершение с просмотром статистики тренировки

## Пример работы бота
Некоторые сообщения бота исчезающие (сообщение о генерации вопроса нейросетью, выбор раздела, выбор следующего действия после ответа), поэтому скриншоты такие.
